## Run on

### Browser

```js
<script src="lib/MultiSprite.js"></script>
<script>
console.log( MultiSprite() );
</script>
```

### WebWorkers

```js
importScripts("lib/MultiSprite.js");

console.log( MultiSprite() );
```

### Node.js

```js
require("lib/MultiSprite.js");

console.log( MultiSprite() );
```

### node-webkit

```js
<script src="lib/MultiSprite.js"></script>
<script>
console.log( MultiSprite() );
</script>
```
